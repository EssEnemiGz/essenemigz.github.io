Github Pages - Rutas

En este artículo se enseñará a usar rutas en Github Pages, además de enseñar a conectarse con otros repositorios del usuario que tengan Github Pages.

Para ponernos en contexto, una ruta sería lo siguiente "essenemigz.github.io/ruta", en este blog se puede apreciar una, la cual es de una carpeta seguida de un archivo; esto se puede extender a repositorios que tengan Github Pages habilitado.

Esto nos puede ser útil para tener una mayor organización en nuestras páginas, o para acceder a contenido estático que ya hayamos alojado en otro repositorio.
